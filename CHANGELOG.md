# Changelog

All notable changes to this project will be documented in this file.

## Version 1.0.0

- Initial release
- Client-side error and rejection tracking
- Event browser with advanced filtering and sorting
- Admin settings for collection configuration
- Automatic data retention and purging
- Group-based access control
- RESTful API with CORS support
- Rate limiting protection
- Built with Svelte 5 and modern TypeScript
